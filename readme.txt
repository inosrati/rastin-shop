== Description ==

Rastin Shop is a lightweight, fast, fully right-to-left (RTL) block theme built for WordPress Full Site Editing and WooCommerce. It ships with a Persian-first design, the Vazirmatn webfont, curated block patterns (hero, featured products, store features) and complete WooCommerce templates (shop, product, cart, checkout, product search).

ویژگی‌ها:

* پشتیبانی کامل از ویرایشگر سایت (FSE) و theme.json نسخه ۳
* راست‌به‌چپ (RTL) و فارسی به‌صورت پیش‌فرض همراه با فونت وزیرمتن
* سازگار با ووکامرس: قالب فروشگاه، صفحه محصول، سبد خرید، تسویه‌حساب و نتایج جستجوی محصول
* الگوهای بلاکی آماده: بخش قهرمان، محصولات ویژه، مزایای فروشگاه
* پالت رنگ، تایپوگرافی سیال و مقیاس فاصله‌گذاری قابل تنظیم
* بهینه برای سرعت؛ بیشتر استایل‌ها از طریق theme.json مدیریت می‌شوند

== Installation ==

1. پوشه `rastin-shop` را در مسیر `wp-content/themes/` قرار دهید.
2. از پیشخوان وردپرس به بخش «نمایش > پوسته‌ها» بروید و «Rastin Shop» را فعال کنید.
3. برای امکانات فروشگاهی، افزونه WooCommerce را نصب و فعال کنید.
4. با «نمایش > ویرایشگر» می‌توانید سربرگ، پابرگ و قالب‌ها را شخصی‌سازی کنید.

== Frequently Asked Questions ==

= آیا این قالب به ووکامرس نیاز دارد؟ =

خیر، قالب بدون ووکامرس هم به‌عنوان یک قالب بلاگ/سایت معمولی کار می‌کند، اما قالب‌ها و الگوهای فروشگاهی تنها با فعال بودن WooCommerce نمایش داده می‌شوند.

= چطور فونت را تغییر دهم؟ =

از بخش «سبک‌ها» در ویرایشگر سایت می‌توانید فونت پیش‌فرض (وزیرمتن) را به فونت سیستمی تغییر دهید.
