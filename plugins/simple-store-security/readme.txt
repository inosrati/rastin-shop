افزونه‌ی امنیتی سبک و کاربردی برای فروشگاه‌های وردپرسی، همراهِ قالب Rastin Shop.

== Description ==

Simple Store Security یک افزونه‌ی امنیتی سبک برای وردپرس و ووکامرس است. همه‌ی قابلیت‌ها از یک صفحه‌ی تنظیمات قابل روشن/خاموش کردن هستند:

* محدودسازی تلاش‌های ناموفق ورود (ضد Brute Force) با قفل ۱۵ دقیقه‌ای پس از ۵ تلاش
* هدرهای امنیتی HTTP (X-Content-Type-Options، X-Frame-Options، Referrer-Policy و ...)
* مخفی‌سازی نسخه‌ی وردپرس و حذف پارامتر ver از استایل/اسکریپت‌ها
* غیرفعال‌سازی XML-RPC و حذف هدر Pingback
* تغییر آدرس صفحه‌ی ورود (Custom Login URL) و مسدودسازی wp-login.php
* جلوگیری از افشای کاربران از طریق REST API و آرشیو نویسنده (author scan)
* پیام عمومی خطای ورود
* غیرفعال‌سازی ویرایشگر فایل پیشخوان
* فایروال ساده در برابر رشته‌های درخواست مشکوک
* گزارش امنیتی در دیتابیس با پاکسازی خودکار روزانه و ایمیل هشدار هنگام قفل شدن IP

== ادغام با قالب Rastin Shop ==

اگر قالب **Rastin Shop** فعال باشد، صفحات این افزونه («امنیت فروشگاه» و «گزارش امنیتی») به‌جای منوی مجزا، زیرِ منوی واحد **«قالب رستین»** در پیشخوان نمایش داده می‌شوند. این کار از طریق ثابت `RASTIN_ADMIN_MENU_SLUG` که قالب تعریف می‌کند انجام می‌شود.

اگر قالب فعال نباشد، افزونه به‌طور مستقل یک منوی سطح‌بالای «امنیت فروشگاه» می‌سازد و کاملاً کار می‌کند.

== Installation ==

1. پوشه‌ی `simple-store-security` را در مسیر `wp-content/plugins/` قرار دهید.
2. از «افزونه‌ها» در پیشخوان، افزونه را فعال کنید.
3. تنظیمات را از منوی «قالب رستین > امنیت فروشگاه» (یا در نبود قالب، منوی «امنیت فروشگاه») انجام دهید.
