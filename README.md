# Rastin Shop

قالب بلاکی (Full Site Editing) فروشگاهی، سبک و کاملاً **راست‌به‌چپ (RTL)** برای وردپرس، سازگار با **WooCommerce** و فارسی به‌صورت پیش‌فرض.

A lightweight, fast, fully RTL WooCommerce-ready WordPress block theme with a Persian-first design and the Vazirmatn webfont.

## ویژگی‌ها

- theme.json نسخه ۳، پالت رنگ، تایپوگرافی سیال و مقیاس فاصله‌گذاری
- فونت وزیرمتن به‌صورت محلی (self-hosted) و پشتیبانی کامل RTL
- قالب‌های محتوا: `index`, `home`, `single`, `page`, `archive`, `search`, `404`
- قالب‌های ووکامرس: فروشگاه، محصول، سبد خرید، تسویه‌حساب، نتایج جستجوی محصول
- الگوهای بلاکی: بخش قهرمان، محصولات ویژه، مزایای فروشگاه
- سربرگ/پابرگ به‌صورت template part

## نصب

1. پوشه‌ی قالب را در `wp-content/themes/rastin-shop/` قرار دهید (یا فایل زیپ را از «نمایش > پوسته‌ها > افزودن» بارگذاری کنید).
2. از «نمایش > پوسته‌ها» قالب **Rastin Shop** را فعال کنید.
3. برای امکانات فروشگاهی، افزونه **WooCommerce** را نصب و فعال کنید.
4. با «نمایش > ویرایشگر» قالب‌ها، سربرگ و پابرگ را شخصی‌سازی کنید.

## نیازمندی‌ها

- WordPress 6.5+
- PHP 7.4+
- WooCommerce 8.0+ (اختیاری، برای امکانات فروشگاهی)
